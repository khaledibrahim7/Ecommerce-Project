# 🍯 Sowar Honey Store - Full Stack Application

تطبيق متكامل للتجارة الإلكترونية لبيع المنتجات الطبيعية وعسل "سوار".

---

## 🛠️ Versions & Tech Stack (التقنيات والنسخ)

### 💻 Frontend:
* **Framework:** Angular 19.2
* **Language:** TypeScript 5.7 & SCSS
* **Build Tool:** Angular CLI 19

### ☕ Backend:
* **Language:** Java 17
* **Framework:** Spring Boot 3.3.5 (Maven)
* **Database:** PostgreSQL 12+
* **Authentication:** JWT (JSON Web Tokens)
* **Documentation:** Swagger UI & OpenAPI 3

---

## 🚀 Main Features (الفيتشرز المتاحة في النظام)

### 👤 الحسابات والأمان (Auth & Security):
* تسجيل حسابات جديدة وتسجيل الدخول بالرقم أو البريد الإلكتروني.
* نظام صلاحيات كامل وفصل للأدوار بين العميل (`CUSTOMER`) والأدمن (`ADMIN`) باستخدام الـ JWT.
* حفظ الـ Token محلياً وتمريره أوتوماتيكياً في الـ Headers مع كل طلب لـ APIs المحمية.
* بروفايل كامل للعميل لتسجيل عناوين التوصيل بالتفصيل.

### 📦 المنتجات والمعرض (Products & Catalog):
* عرض المنتجات مقسمة حسب الفئات (Categories) مع دعم الـ Pagination والـ Sorting في البحث.
* صفحة تفاصيل لكل منتج تدعم تقييمات العملاء، المراجعات، ونظام الأسئلة والأجوبة (Q&A).
* نظام عروض متكامل (خصومات مباشرة على الأسعار أو عروض المنتجات الهدايا).
* لوحة تحكم للأدمن لإضافة، تعديل، وحذف المنتجات والفئات، وإدارة المخزون وصور المنتجات.

### 🛒 السلة والطلبات (Cart & Orders):
* سلة مشتريات (Cart) ديناميكية لحفظ المنتجات وتعديل الكميات وحساب الإجمالي.
* قائمة المفضلة (Wishlist) لحفظ المنتجات.
* دورة طلب كاملة (Order Flow) تدعم الشحن للمحافظات والدفع عند الاستلام (COD).
* حساب مصاريف الشحن أوتوماتيكياً بناءً على محافظة العميل.
* شريط زمني لتتبع حالة الطلب خطوة بخطوة (Order Status Timeline) للعميل وللأدمن.
* أخذ لقطة (Snapshot) من سعر المنتج وتفاصيل العرض وقت الشراء لضمان ثبات الحسابات المالية.

### 📊 الإدارة والتقارير (Admin & Finance):
* لوحة تحكم (Dashboard) كاملة لمتابعة الطلبات وتحديث حالاتها.
* نظام لتسجيل المصاريف اليدوية والنثرية (Manual Expenses).
* تقارير مالية تفصيلية للإيرادات وصافي الأرباح مقسمة شهرياً وسنوياً.
* نظام إرسال إشعارات عبر البريد الإلكتروني لتأكيد التسجيل وتحديثات الطلبات.
