# Sowar Honey Store (Backend) 🍯

## 🛠️ Versions & Tech Stack
* **Java:** 17
* **Framework:** Spring Boot 3.x (Maven)
* **Database:** PostgreSQL
* **Security:** JWT Authentication

---

## 🚀 Main Features 

### 👥 Auth & Users:
* تسجيل وحسابات للعملاء (Customer) مع إمكانية اللوجن بالإيميل أو رقم الموبايل.
* نظام صلاحيات كامل وفصل للأدوار بين الـ `CUSTOMER` والـ `ADMIN` باستخدام الـ JWT.
* بروفايل للعميل لتسجيل العناوين بالتفصيل وبيانات التوصيل.

### 📦 Products & Catalog:
* إضافة منتجات منقسمة لفئات (Categories)، مع دعم رفع صور متعددة وتتبع كميات المخزون.
* نظام عروض متكامل (خصم مباشر على السعر أو عرض قطعتين ومعاهم قطعة هدية).
* نظام مراجعات، تقييمات، وأسئلة وأجوبة (Q&A) على المنتجات.
* بحث متقدم في المنتجات مع الـ Pagination والـ Sorting.

### 🛒 Cart & Orders:
* سلة مشتريات (Cart) لحفظ المنتجات والكاونتر وتجهيز الـ Checkout.
* دورة طلب كاملة (Order Flow) بتدعم الدفع عند الاستلام (COD).
* حساب مصاريف الشحن أوتوماتيكياً بناءً على محافظة العميل.
* تتبع حالة الطلب خطوة بخطوة للعميل وللأدمن عبر (Order Status Timeline).
* حفظ لقطة (Snapshot) من سعر المنتج والعرض وقت الشراء عشان الحسابات ما تضربش لو السعر اتغير بعدين.
* قائمة الأمنيات (Wishlist).

### 📊 Admin & Finance:
* لوحة تحكم كاملة للأدمن لإدارة المنتجات، الفئات، والطلبات والمراجعات.
* تسجيل المصاريف النثرية واليدوية (Manual Expenses).
* تقارير مالية جاهزة للإيرادات والأرباح (صافي ومجمل) مقسمة شهرياً وسنوياً.
