How to Run


### 💻 1. تشغيل frontend (Angular)
افتح الـ Terminal جوة فولدر الـ Frontend واكتب:

npm install
npm start
التطبيق هيكون متاح على: http://localhost:4200



☕ 2. تشغيل backend (Spring Boot)
افتح الـ Terminal جوة فولدر الـ Backend واكتب:

Bash
mvn clean install -DskipTests
mvn spring-boot:run
السيرفر هيكون متاح على بورت 8080.
التوثيق (Swagger) متاح على: http://localhost:8080/swagger-ui.html
